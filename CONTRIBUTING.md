# Configurar el entorno de desarrollo

### Prerrequisitos

Clone el repositorio. Desde la terminal, corra:

``` bash
git clone https://github.com/Sofoclesias/music-source-separation.git
```

A continuación, diríjase a la carpeta clonada, cree un entorno virtual y ejecuta una instalación editable de librería.

``` bash
cd music-source-separation/
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -e .
``` 

Ahora ya puede realizar ediciones y ejecutar comandos con el formato organizado de librería de audiomancer.

``` python
import audiomancy
```

### Dependencias externas

Para obtener los códecs de los audios y realizar las operaciones con librosa, se debe instalar el paquete FFmpeg y FFprobe.
Luego, para la confección de mixes aleatorios mediante los archivos .jams, es necesario tener instalado también el programa SoX y descargar la carpeta de stems del repositorio.

FFmpeg y FFprobe
- Instalar con el gestor de paquetes de línea de comandos para Microsoft Windows, chocolatey. (Recomendado).
``` bash
choco install ffmpeg
```
- Descargar de la fuente original: https://ffmpeg.org//download.html

Sox:
- Instalar con chocolatey. (Recomendado, así se me configuró bien a mí.)
``` bash
import audiomancy
audiomancy.common.download_stems()
```
- Descargar de la fuente original: http://sox.sourceforge.net/

Stems:
- Instalar desde la misma librería de audiomancy.
``` python
import audiomancy
audiomancy.common.download_stems()
```
- Descargar desde el enlace directo de MEGA: https://mega.nz/file/eUwHCLCJ#7g4qRZnCxxgnQyY8WzWaWAjg14k_D59FiJRjLyz1MJo
